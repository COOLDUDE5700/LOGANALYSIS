The code is contained in a single file : log-analysis.py Python 2.7.12 was used during development.

The purpose is to use Python and PostgreSQL to build an internal reporting tool to query databases and provide answers to questions about authors, articles, and links of a news website.

To run it, execute python log-analysis.py or, if the file has execution permissions, execute ./log-analysis.py. This assumes you're in the directory where this file is located.
